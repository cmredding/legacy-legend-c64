C64 cartridge app · key command build
New: command buttons now send C64 keyboard letters. If commands do not respond, open ☰ → Keyboard Support once, then tap the game screen.

Command buttons

  • The bottom buttons send normal C64 keyboard letters: A, F, G, H, I, P, R, S, T, U, W, X.
  • If a button does nothing, tap ☰ → Keyboard Support once, then tap the game screen and try again.
  • D-pad and Fire are still joystick controls.
  • Return confirms prompts. Del erases during typing.
Android blocks some keyboard routing until the emulator's own Keyboard Support mode is enabled. The buttons are now linked to keyboard events, not gamepad buttons.
